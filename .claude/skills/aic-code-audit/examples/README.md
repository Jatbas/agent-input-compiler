## What the example should show

When authored, the example must include:

- A realistic Phase BUGS table with at least one Critical (Next Skill: `aic-systematic-debugging`),
  one High chain (Next Skill: `aic-task-planner`), and one Medium entry.
- The corresponding header block update at the top of `aic-progress.md`.
- At least one discarded false positive with reasoning.

The example must pass `evidence-scan.sh` — every finding cites `file:line`.
