METHOD BEHAVIORS:

- transform: Returns content unchanged if length is 0. Otherwise parses CSS/SCSS into rule blocks (brace-matched). For each block: if selector is :root, keep the block and collapse newlines to a single space; for all other blocks, keep the selector (and any at-rule header) and replace the declaration body with " [N declarations] " where N is the number of declarations (semicolon-terminated) in the body. Concatenates processed blocks and any inter-block whitespace/comments that are not inside a block, preserving order. Returns the resulting string.
