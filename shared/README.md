# @jatbas/aic-core

Internal core library for [AIC (Agent Input Compiler)](https://github.com/Jatbas/agent-input-compiler).

This package contains the compilation pipeline, storage layer, adapters, and shared types used by `@jatbas/aic`. It is not intended for direct use — install `@jatbas/aic` instead.
